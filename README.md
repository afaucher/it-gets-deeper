# It Gets Deeper

A procedural 3D tunnel flyer built in Godot 4.

## Getting Started

### Prerequisites
- Godot 4.x
- Git

### Installation
1.  Clone the repository.
2.  Open the project in Godot.

### How to Run
- Run `run_game.ps1` in PowerShell.
- Or open the project in Godot and press F5 (Play).

## Controls
- **WASD / Arrow Keys**: Steer ship (Up/Down/Left/Right).
- **Space**: Fire weapon.
- **Escape**: Quit game.

## Gameplay
- Fly down the infinite tunnel.
- Avoid walls (bouncing causes damage/speed loss).
- Shoot enemies (red ships) flying up at you.
- Manage your **Ammo**: Reloads automatically but takes time.
- Survive as long as possible to increase your **Score**.
